# Dokumentasi Core Banking BPR

Selamat datang di dokumentasi resmi **Core Banking BPR** - sistem informasi perbankan terintegrasi untuk Bank Perkreditan Rakyat.

## Tentang Aplikasi

Core Banking BPR adalah aplikasi manajemen perbankan yang mencakup seluruh operasional BPR, mulai dari pengelolaan data nasabah, produk simpanan & kredit, transaksi teller, hingga akuntansi dan pelaporan.

## Modul Utama

| Modul | Deskripsi |
|-------|-----------|
| **Master Data** | Pengelolaan data cabang, nasabah, bagan akun, produk tabungan, deposito, dan kredit |
| **Teller** | Operasional teller harian: setor, tarik, bayar angsuran |
| **Kredit** | Permohonan kredit, persetujuan, pencairan, dan monitoring |
| **Akuntansi** | Jurnal, neraca saldo, neraca, dan laporan laba rugi |
| **Operasional** | Proses End of Day (EOD) |
| **Laporan** | Portofolio kredit dan laporan manajemen |
| **Administrasi** | Manajemen pengguna, peran, dan parameter sistem |
